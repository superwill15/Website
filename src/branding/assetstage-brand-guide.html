Usage Guidelines

  • Logo Clear Space: Maintain minimum 1/2 logo height on all sides Important
  • Minimum Sizes: Digital: 120px width | Print: 1 inch | App icon: 16px
  • Color Usage: Primary navy for headers, stage blue for CTAs, green for success states
  • Typography Hierarchy: Use consistent heading sizes: 48px → 32px → 24px → 16px
  • Dark Mode: Switch to reverse logo, adjust text colors for contrast New
  • Loading States: Use rotating checkmark animation from logo icon
  • Error States: Replace green checkmark with red X, maintain icon structure
  • Mobile Responsive: Switch to icon-only below 768px viewport width
